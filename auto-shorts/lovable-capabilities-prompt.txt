GOAL
Add three new AI features to each generated short — A/B hook variations, a
virality score, and a cover/thumbnail concept — using Lovable Cloud edge
functions with Lovable AI. Keep everything inside Lovable.

1) Edge function `hook-variations`
   - Input: { plan, count?: number (default 3, max 8) }
   - Use Lovable AI to return { hooks: string[] } — `count` distinct alternative
     hooks (<=12 words each), varying the angle (curiosity, bold claim, question,
     stat, contrarian).

2) Edge function `virality-score`
   - Input: { plan }
   - Use Lovable AI to return:
     { score: number(0-100),
       breakdown: { hook:0-100, pacing:0-100, payoff:0-100, shareability:0-100 },
       reasons: string[] (<=5) }
   - Clamp all numbers to 0-100.

3) Edge function `cover-concept`
   - Input: { plan, brandColor?: string }
   - Use Lovable AI to return:
     { coverText: string(<=5 words), subText: string, backgroundIdea: string,
       emoji: string, textColor: "#RRGGBB" }
   - If textColor isn't a valid hex, use brandColor (if valid) else "#FFFFFF".

4) Frontend wiring (src/lib/api.ts) — add typed callers for the three functions
   (functions.invoke), each with a graceful fallback on error.

5) UI surfaces
   - ShortCard: show a small VIRALITY badge (e.g. "🔥 82") on each card. Fetch it
     lazily (after the grid renders, or on a "Score" tap). Colour the badge by
     band: >=75 green, 50-74 amber, <50 slate.
   - EditModal: add two sections:
     a) "A/B Hooks" — a button that calls hook-variations and lists the returned
        hooks; tapping one sets it as the short's hook.
     b) "Cover" — a button that calls cover-concept and renders a 9:16 preview
        tile: backgroundIdea as a subtle caption, coverText big (textColor),
        subText small, and the emoji; plus a "Copy cover text" action.

6) Keep all existing behaviour (generate, variation, demo fallback) intact.

ACCEPTANCE
- Each card shows a virality badge from real AI.
- The edit modal can generate alternative hooks (and apply one) and a cover
  concept preview.
- No external backend, API key, or hosting — all via Lovable Cloud + Lovable AI.
